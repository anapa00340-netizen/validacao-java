# Sistema de Validação de Dados em Java

Projeto desenvolvido em Java com o objetivo de realizar cadastro de pessoas com validação de dados.

## Funcionalidades
- Cadastro de usuários
- Validaçao de :
- Nome
- E-mail
- CPF
- Exibiçao de relatório com todas as pessoas cadastradas

## Tecnologias utilizadas
- Java
- Eclipse

- ## Objetivo do projeto
Projeto desenvolvido para praticarlógica de programação, validação de dados e estruturas de repetição em Java.
